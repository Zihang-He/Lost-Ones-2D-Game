## 🧰 环境与安装
### 1. 安装 Unity
- 使用 **Unity Hub**
- 版本号：**2022.3.34f1c1**
- 平台：macOS 或 Windows 都可  
  - 如果两人系统不同，版本号保持一致即可（Unity 项目是跨平台的）

### 2. 克隆项目
```bash
git clone https://github.com/Zihang-He/Lost-Ones-2D-Game.git
cd Lost-Ones-2D-Game
git lfs install
git lfs pull

Assets/
├─ Scripts/ # 游戏逻辑脚本 (ActRunner.cs 等)
├─ Data/ # 剧情文件，例如 story.json
├─ Prefabs/ # 可复用的 UI 元件，例如按钮预制体
├─ Resources/
│ └─ Art/ # 背景图资源 (lake.png, cabin.png ...)
└─ Scenes/ # 主场景 (Main.unity)

Theme: Quelque secondes, une éternité (some seconds, an eternity)

一个typically successful的人在一场巨大的成功后，平静地走进了自己的办公室，用自己的领带吊死了自己。

他是一个能力很强的人，在大多数时间都可以控制自己的情绪，这也是为什么他能够成为一个成功的人，但是他其实长期患有双相情感障碍，而人群的在他身边的过度聚集就是一个trigger，他会因此进入抑郁或者躁狂状态，而我们需要展示他在这些痛苦中的挣扎。


Scene 0（办公室）:
镜头：视角先上移再快速下移很短的一段，然后注视10秒钟时间柜子和钟，最后渐渐闭上眼睛。
内容：屏幕部分环绕着红色的领带，中间是一个座钟，中间靠左是一个可以交互打开其中一个储物格的立式书柜。
可操作部分：在10秒钟以内可以点击交互柜子打开看到药瓶子，可以点击。
素材：柜子、打开的柜子、座钟、墙。

Scene 1（工位）（抑郁）：
镜头：首先出现的是一个电脑界面，在完成第一步操作后视角zoom out，看到一整个工位；此时一只手拿着奖杯移动到桌子上，工位隔板后面渐显人，几秒后视野开始模糊，鼠标开始抖动，渐渐变成灰度，完成第二步操作后视角zoom out，看到一整个工位，完成第三步操作后视野渐渐恢复颜色且zoom in到钟表，进入转场。
声效：一开始是正常的音乐，随着人群的渐显嘈杂声渐渐增大，然后随着视线模糊的加重耳鸣声渐渐增大，呼吸声渐渐增大；随着第三步操作渐渐恢复音乐。
内容：工位上的陈设自左向右为：显示器、小钟、奖杯、小柜子；电脑界面的内容为左边两个文件夹，标题分别为“Finished Tasks”“Unfinished Tasks”；右边两个文件，标题分别为“Finished Task”“Unfinished Task”，中间两行字：“Organise the tasks”
素材：电脑界面（！）、显示器、奖杯、工位大致框架、人、小钟、小柜子+打开的小柜子、药瓶；嘈杂的声音、呼吸声、耳鸣声、普通的音乐。
第一步操作：将“Finished Task”拖动到“Finished Tasks”里面。
第二步操作：将“Unfinished Task”拖动到“Unfinished Tasks”里面。
第三步操作：点击贴了红色标的柜门，再点击里面的药盒。
其中，可以直接进行第三步操作来跳过这一幕。
Scene 2（游乐园）（躁狂）：
镜头：首先出现的是一个游乐园的场景，第一步操作后视角zoom in到售票处，一只手拿着票从下往上移动；完成第二步操作以后视角zoom out，人群出现（trigger）；画面对比度升高，边缘更锐利（不确定这个滤镜能不能做）此时远处过山车断裂（Ich bin übermensch），第1、4、5、7个人掉下来（过山车需要8节）；完成第三步操作后视角zoom in 到售票处，伸出来一只手拿着之前的票；完成第四步操作后手托着药盒子出来（不是药瓶子）视角移动到售票处上方，注视时钟，渐渐恢复正常，转场。
声效：一开始是正常的音乐，人群出现以后配一个欢声笑语的声效，躁狂期间改成躁狂的音乐，之后再回到原来的。
内容：游乐园的场景主要为前景一个大门，左边是售票处，背景是巨大的过山车。售票处中间是漆黑的窗口，别的地方贴了一些海报（可以没有），上方是一个时钟。
素材：过山车、整体场景、售票处、人影、掉落小人、药盒、手、票；欢声笑语、配乐。
第一步操作：点击售票处。
第二步操作：点击票上面的二维码。
第三步操作：把掉下来的人拖到过山车上。
第四步操作：点击票上对应的1、4、5、7。
其中，可以在人掉下来的时候直接点击售票处来跳过这一幕。
Scene 3 （厨房）（两种）（可以先skip，这个做不完就不做）：
镜头：（这个部分可以借鉴Edith Finch）首先出现的是切鱼

Scene 3（礼堂）（直接原因）：
镜头：首先是一个空旷的礼堂，完成第一步操作以后时间走动一秒，人群渐显，眼前出现一个巨大的奖杯；完成第二步操作以后屏幕变黑，画面变为黑白，视角切换到碎掉的奖杯；完成第三步操作以后随之前的操作出现药盒子/药瓶子/完整的奖杯（Und Zarathustra will wieder Mensch werden），然后视野变黑，重新回到钟，画面正常。
声效：一开始是轻微的嘈杂声，人群渐显时声音变大，完成第二步操作后配躁狂音乐，完成第三步操作后变成普通音乐。
内容：空旷的礼堂，主色调为米色，用几条线表示座位，画面下方是一个讲台（？），画面上方挂了一个钟，钟一开始比之前的钟要早一格。
素材：礼堂、奖杯、碎掉的奖杯、人影、声效。
第一步操作：点击时钟。
第二步操作：点击奖杯。
第三步操作：拼奖杯。
之前的操作与出现的事情：
在第一幕里点击了柜子里的药同时跳过了Scene：药盒子。
没有在第一幕里点击了柜子里的药但同时跳过了Scene：药瓶子。
完成了所有Scene的工作：完整的奖杯。


Scene 4（办公室）：
根据Scene 4的情况，出现A、B、C三种结局。
A（出现药盒子）：和Scene 0一模一样的场景，但是钟表开始走动，然后眼睛慢慢闭上。
B（出现完整的奖杯）：和Scene 0一模一样的场景，但是钟表开始走动，视角迅速下落，切换到一个人躺在地上的样子，声效切到轻微的笑声。
C（出现药瓶子）：看到一个挂着的领带圈，然后视角下移渐渐变黑。
素材：领带圈、一个人躺在地上的样子。
